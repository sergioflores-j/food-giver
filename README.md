## Dependências externas

Essas dependências devem estar previamente instaladas na máquina antes de começar a interagir com o projeto.

- Node v12.x+
- Java v8+

## Primeiros passos

- Instalando as dependências:
```bash
npm run install-all
```

- Iniciando os serviços e a aplicação
```bash
npm run all
```
